FUME

Flexible Universal processor for Modeling Emissions

View on GitHub

The first version of emission processor FUME (Flexible Universal processor for Modeling Emissions) was created within the project of the Technology Agency of the Czech Republic No. TA04020797 Advanced emission processor utilizing new data sources. Software is distributed free of charge under the licence GNU GPL v3.0. With requests for technical support please contact Mr. Ondrej Vlcek.
